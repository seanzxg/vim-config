### leader快捷键

| 命令          | 描述             |
| ------------- | ---------------- |
| <leader>q     | 缓存区退出       |
| <leader>sv/ss | vsplit或者split  |
| <leader>r     | coc rename       |
| <leader>a     | 当前缓存区action |
| <leader>`     | 修复当前行       |
| <leader>g     | git操作          |
| <leader>u     | undotree         |
| <leader>\     | 命令行           |
| <leader>v     | 粘贴剪切板       |
